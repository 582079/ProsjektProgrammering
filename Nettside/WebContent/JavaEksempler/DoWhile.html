DoWhile

Dette blir et litt mer komplisert eksempel, men hvis du bare vil vite om selve løkken kommer den tidelig.
Vi starter med et skjellet og importerer java.lang.Integer.* og javax.swing.JOptionPane.*.
Vi skal veksle penger for et produkt verdt 200 kr.
Det "do while"-løkken vil gjøre her er å sjekke om verdien gitt er gyldig.
Først starter vi med noen variabler.

int pris = 200;
int antallKr;
boolean gyldigAntall = false;
int rest = 0; // Resten av vekslepengene
int[] krEnheter = {1, 5, 10, 20, 50, 100, 200, 500, 1000};
int[] antallEnheter = new int[krEnheter.length]; // Hvor mange av hver enhet

Vi har pris og antall betalt.
For å huske om vi har et gyldig antall betalt må vi ha en sannhetverdi gyldigAntall
Til veksling har vi en rest, som holder orden på hvor mange av pengene som ikke er vekslet enda.
Vi har tabeller som vi lar oss trekke penger fra rest med en løkke og liste opp antallet av hver enhet.

Vi begynner med en tom "do while"-løkke:

do {

} while(!gyldigAntall)

I motsetning til en vanlig while-løkke, står while og sannhetsverdien på slutten av parentesene.
Dette er siden while sjekker om det er et gyldig antall etter vi har kjørt koden.
Det vil si at vi alltid går gjennom løkken minst en gang.
Vi er også interisserte i å sjekke om antallet er gyldig først etter vi har fått en verdi, hvis den er det er det ikke noe mer som vi trenger å spør etter.

Vi gir antallKr en verdi som blir omgjort fra Stringen vi får fra showInputDialog().
Vi gir også en melding som forklarer hvorfor brukeren må gi en ny verdi, om de skulle skrive en feil verdi først.

antallKr = parseInt(showInputDialog("Gi et antall kr over " + pris + " kr: "));

Nå sjekker vi om det er en gyldig verdi:

if(antallKr >= pris) {
				
	gyldigAntall = true;
				
}

Nå kommer vi tilbake til while-delen, som spør om den gitte verdien var gyldig.

while(!gyldigAntall)

Var den det, så går vi ut av løkken og fortsetter.
Var den ikke gyldig, fortsetter "do while"-løkken.
Det vil si at vi burde komme ut med en gyldig verdi etter hvert.

Nå ser vi hvor mye vi hvor mange kr vi må gi tilbake.

rest = antallKr - pris;

Vi starter en for-løkke for å gå gjennom alle kroneenhetene og finne ut hvor mange av hver enhet vi skal gi. Vi begynner med en tom for-løkke:

for(int i = krEnheter.length-1; i>=0; i--) {

}

Merk to ting:
Vi sier at indeksen er en mindre enn lengden. Dette er siden den høyeste indeksen i en tabell er en mindre enn lengden.
Vi begynner også fra den høyeste indeksen og går ned null.
Dette er siden vi vil veksle de største krone-enhetene først, ellers ville vi bare gi enkroninger.

Vi spør om resten delt på den minste krone-enheten hittil er større enn null:

if(rest / krEnheter[i] > 0) {

}

Husk at dette er heltallsdivisjon, så med mindre resten er større enn minst en ganger krone-enheten, vil dette ikke være sant.
Hvis det er sant utfører vi denne koden:

antallEnheter[i] = rest / krEnheter[i];
rest = rest % krEnheter[i];

Verdien i antallEnheter med samme indeks vil bli gitt denne heltalls-divisjonen, som er lik antallet av enheten.
Verdien vi gir rest bruker et prosenttegn. Dette kalles "modulo" her, og gir resten hvis du deler den venstre siden på den høyre siden.

Etter if, men innenfor for-løkken, skriver vi ut en linje som oppsummerer antallet av enheten vi jobber med.

System.out.println("Antall " + krEnheter[i] + "kr : " + antallEnheter[i]);

Nå burde det fungere og koden burde se omtrent slik ut:

package pakke;

import static java.lang.Integer.*;
import static javax.swing.JOptionPane.*;

public class DoWhile {

	public static void main(String[] args) {
		
		int pris = 200;
		int antallKr;
		boolean gyldigAntall = false;
		int rest = 0; // Resten av vekslepengene
		int[] krEnheter = {1, 5, 10, 20, 50, 100, 200, 500, 1000};
		int[] antallEnheter = new int[krEnheter.length]; // Hvor mange av hver enhet
		
		do {
			
			antallKr = parseInt(showInputDialog("Gi et antall kr over " + pris + " kr: "));
			
			if(antallKr >= pris) {
				
				gyldigAntall = true;
				
			}
			
		} while (!gyldigAntall);
		
		// Gi vekslepenger
		rest = antallKr - pris;
		// Gå gjennom krone-enheter, og begynn med å veksle de største
		for(int i = krEnheter.length-1; i>=0; i--) {
			
			if(rest / krEnheter[i] > 0) {
				
				antallEnheter[i] = rest / krEnheter[i];
				rest = rest % krEnheter[i];
				
			}
			
			System.out.println("Antall " + krEnheter[i] + "kr : " + antallEnheter[i]);
			
		}

	}

}
