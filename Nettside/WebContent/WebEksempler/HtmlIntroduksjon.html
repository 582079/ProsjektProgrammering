Introduksjon

HTML(Hyper Text Markup Language) er ett standard markup språk for å lage nettsider.
for å begynne med å lage en nettside så kan du bruke en tekst editor som Notepad eller Notepad ++.
Du må huske å lagre dokumentet som ett HTML dokument sånn at nettleseren som du bruker kan lese dokumente.
selv omdu kan bruke en vanlig teks editor så kan du også bruke en IDE(Integrert Utviklermiljø) som for eksempel "Eclipse"
En IDE kan gjøre dokumentet ditt mer oversiklig og mer leslig.
Ett html dokument består hovedsaklig av start tagger pluss slutt tagger og elementer som er byggeklossene til html sider
dom blir representert av tagger.
Meningen til nettleseren er å lese og visualisere et html dokument
Nettleseren visualiserer ikke HTML taggene, men bruker de til å fastslå hvordan dokumentet skal vises

Hoved Elementer til ett html dokument

  • "<!DOCTYPE html>" som kunngjør at dette er ett Html5 dokument.
  • "<html>" er hoved elementet til en html side.
  • "<head>" elementet inneholder informasjon om dokumentet.
  • "<title>" elementet spesifiserer en titel for dokumentet.
  • "<body>" inneholder alt som blir synlig på nettsiden.

Eksempel på ett html dokument: 

  <!DOCTYPE html>
     <html>
      <head>
       <title> Side Titel <title>
        </head>
        <body>

        <h1> Min første Overskrift </h1>
        <p>Min første Paragraf </p>

         </body>
          </html>

Det er to elementer i ett HTML dokument du kan bruke for å visualisere tekst på nettsiden.
Vi har Paragraf elementet <p> denne brukes til å skrive simpel tekst på nettsiden,
og vi har Header elementet <h1> til <h6>. Denne kan brukes til å legge inn mer viktig tekst
som overskrift på nettsiden der <h1> representerer mer viktig tekst og opp til <h6> til mindre viktig tekst.
