HTML Styling

For å sette litt stil på nettsiden så kan man bruke Style egenskapen til ett HTML dokument.
Styling egenskapen er fra CSS(Cascading Style Sheet) som vi lærer mer om seinere i denne opplæringen.
Her er noen av de mest vanligst Styling egenskapen vi bruker når vi designer en nettside.

HTML Bakgrunn farge

"background-color" definerer bakgrunns fargen til ett HTML element.
Her er ett eksempel på ett HTML dokument som bruker denne egenskapen. 

 
<body style="background-color:powderblue;">

<h1>Overskrift</h1>
<p>Paragraf.</p>

</body>

HTML Tekst Farge

"color" definerer tekst fargen til ett HTML element
her er ett eksempel på ett HTML dokument som bruker denne egenskapen. 

<h1 style="color:blue;">Overskrift</h1>
<p style="color:red;">Paragraf.</p>

HTML Fonts og Tekst Størrelse

"font-family" definerer font(tekst Stil) til ett HTML dokument.
Eksempel på ett HTML dokument som bruker denne egenskapen. 

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

Vi har også en egenskap som definerer tekst størrelse på ett HTML dokument
som er "font-size".
Her er ett eksempel på ett HTML dokument som bruker denne egenskapen. 

<h1 style="font-size:300%;">Overskrift</h1>
<p style="font-size:160%;">Paragraf.</p>

Oppsumering

  • Bruker "style" egenskapen for å få en mer stilfull nettside
  • Bruker "background-color" for bakgrunn farge
  • Bruker "color" for tekst farge
  • bruker "font-family" for å få en mer stilfull tekst
  • bruker "font-size" for tekst størrelse
